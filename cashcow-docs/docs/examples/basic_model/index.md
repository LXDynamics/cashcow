# Basic Model Example

This directory contains a complete starter configuration for a rocket engine startup company using CashCow.

## Files Included

- `config.yaml` - Main configuration file
- `entities/` - Sample entity definitions
  - `employees/founder-ceo.yaml` - Founder/CEO with equity
  - `grants/sbir-phase1.yaml` - NSF SBIR Phase I grant  
  - `facilities/startup-office.yaml` - Headquarters and lab facility

## Usage

Copy these files to your CashCow project directory and customize as needed for your specific company structure.
